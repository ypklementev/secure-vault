# Secure Vault

## Описание

Secure Vault --- это веб-приложение для локального шифрования и
дешифрования файлов в браузере с использованием алгоритма AES-GCM. Все
операции выполняются на стороне клиента без отправки данных на сервер.

## Возможности

-   Генерация ключа шифрования (256-bit)
-   Шифрование файлов
-   Дешифрование файлов
-   Поддержка нескольких файлов
-   Drag & Drop интерфейс

## Использование

### 1. Генерация ключа

Нажмите кнопку "Создать ключ" и сохраните файл `encryption.key`.

### 2. Загрузка ключа

Нажмите "Выбрать ключ" и выберите ранее сохранённый файл ключа.

### 3. Добавление файлов

Перетащите файлы в область загрузки или выберите их через диалог.

### 4. Шифрование

Нажмите "Шифровать". Файлы будут сохранены с расширением `.enc`.

### 5. Дешифрование

Добавьте зашифрованные файлы (`.enc`) и нажмите "Дешифровать".

## Требования

-   Современный браузер с поддержкой Web Crypto API

## Примечания

-   Ключ хранится только в рамках текущей сессии
-   Потеря ключа делает невозможным восстановление данных
-   Все операции выполняются локально
